# dartic_cli

Command-line interface for the dartic bytecode interpreter.

## Commands

- `dartic compile` -- Compile a Dart kernel (.dill) to dartic bytecode (.darb)
- `dartic run` -- Execute a .darb bytecode file
- `dartic dump` -- Disassemble and inspect a .darb file
- `dartic gen` -- Generate bridge bindings from YAML configuration
- `dartic doctor` -- Check environment setup
- `dartic version` -- Print version information
